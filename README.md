# NitoriToolBox

## 这是什么？

NitoriToolbox是一个开放式应用工具箱，通过插件式便捷地将应用、应用所需要的环境导入导出，并提供类似于Jupyter Notebook的方式来运行应用

## 制作插件

请参考 Wiki，如有意见欢迎提出 Issue

https://github.com/CCZU-OSSA/NitoriToolbox/wiki
